## Quick Diagnostic Commands

```bash
# 1. Check if backend is running
curl http://localhost:3000/health
# Expected: {"status":"OK",...}

# 2. Check alerts in database
docker exec -it ecocheck-postgres psql -U ecocheck_user -d ecocheck \
  -c "SELECT COUNT(*) FROM alerts;"
# Expected: 0 (that's why no alerts show)

# 3. Check API response
curl http://localhost:3000/api/alerts
# Expected: {"ok":true,"data":[]}

# 4. Create test route
curl -X POST http://localhost:3000/api/test/start-route \
  -H "Content-Type: application/json" \
  -d '{"route_id":"test-1","vehicle_id":"V01"}'
# Expected: {"ok":true,"message":"Test route...","points":[...]}

# 5. Wait and check again
sleep 20
curl http://localhost:3000/api/alerts
# Expected: {"ok":true,"data":[{...}]}
```
